FFMpeg-for-ANDROID-with-NDK
===========================

##Environments
* Windows 7 32bit
* android-ndk-r8d
* cygwin

## NDK Download & Setting
* Download : http://developer.android.com/intl/ko/tools/sdk/ndk/index.html
* Setting : http://developer.android.com/intl/ko/tools/sdk/ndk/index.html#GetStarted

##Build
```
cd $prj
ndk-build -j4
```
